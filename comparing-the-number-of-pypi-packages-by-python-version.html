Comparing the number of PyPi packages by Python version

Posted on Sat 07 January 2017 in Python

I periodically collected the number of PyPi packages from their site to get insight to the current popularity of Python2 vs. Python3.

Here are the results:

Technical details

Data scraper script:

from sys import stdout
from time import sleep, time
import requests
from lxml import html


def fetch():
    while 1:
        resp = requests.get('https://pypi.python.org/pypi?%3Aaction=browse')
        dom = html.fromstring(resp.text.encode('utf-8'))
        stdout.write('timestamp: {}, '.format(int(time())))
        print(', '.join(': '.join(x.strip() for x in v.xpath('.//text()') if x.strip())
                        for v in dom.xpath('//*[text()="Programming Language :: Python"]/../ul[1]/li')))
        stdout.flush()
        sleep(120)
